# TUI Hackernews

## Description

TUI Hackernews is a terminal based client for news.ycombinator.com (hackernews) written in c

## Dependencies

- libcurl
- ncurses
- cjson
- firefox

## Install and Run

Install dependencies and then:
```
git clone https://github.com/jackhipson300/tui-hackernews.git
cd tui-hackernews
make production
./hnews
```
